## Overview

You're looking at it. A minimal, fast personal site built with Astro for sharing my work and writing.

## Goals

- Fast and accessible (no bloat)
- Easy to write new posts in Markdown
- Clean dark design that gets out of the way
- Deploy to Vercel with zero config

## Stack

- **Astro** — static site generation
- **TypeScript** — content collection schemas
- **CSS** — no framework, just custom properties
- **Vercel** — hosting and deploys

## Lessons

Building your own site is a great forcing function for shipping. Every decision is yours — no excuses.
